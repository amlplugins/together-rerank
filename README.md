# @amlplugins/together-rerank

Native `together-ai` SDK re-exported under the `@amlplugins` namespace.

> Together Rerank — Salesforce LlamaRank cross-encoder reranking for RAG pipelines.

## Install

```bash
npm install @amlplugins/together-rerank
```

## Use

```ts
import { /* named exports from together-ai */ } from "@amlplugins/together-rerank";
// or
import sdk from "@amlplugins/together-rerank";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
